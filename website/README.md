# Super Bash Folds website

This directory contains the public landing page, credits, and hosted game
bundle for Super Bash Folds.

## Local development

Prerequisite: Node.js 22.13 or newer.

```sh
cd ..
npm ci
npm run site:sync-game
cd website
npm ci
npm run dev
```

Run verification with:

```sh
npm test
```

The game under `public/play/` is generated by `site:sync-game` and is not edited
by hand or committed. Re-run the root command whenever the game changes:

```sh
cd ..
npm run site:sync-game
```

The deployed site is public and must contain only original or redistributable
assets. Credits and source links live at `/credits`.
